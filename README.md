# statiko-public

[Statiko](https://statiko.io) lets people search and browse Telegram channels, track their subscriber/activity stats over time, and view a public timeline of channel events (post edits, deletions, metadata changes, subscriber milestones). Signed-in users can bookmark channels, group them into collections, and get recaps.

This is a public repository to share [feature requests](https://github.com/marfer/statiko-public/issues/new?template=feature_request.md), [bug reports](https://github.com/marfer/statiko-public/issues/new?template=bug_report.md), and [roadmap discussions](https://github.com/users/marfer/projects/2/views/1).
